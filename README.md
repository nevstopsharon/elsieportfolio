# WEI XINRU Portfolio

Static portfolio website for WEI XINRU.

## Local preview

```powershell
py -3 -m http.server 8765 --bind 127.0.0.1
```

Open `http://127.0.0.1:8765/`.

## Content updates

Edit `site-content.js` for project copy, social links, photography eras and contact details. The current release is a visual prototype and intentionally remains `noindex` until verified content is complete.
